# Thiệp cưới Khương Đỗ 💗 Quỳnh Anh

Website thiệp cưới (25/09/2026). Trang tĩnh, chỉ gồm `index.html` + `anh.jpg`.

## Xem tại chỗ
Mở thẳng `index.html` bằng trình duyệt, hoặc chạy:
```
python3 -m http.server 8777
```
rồi mở http://localhost:8777

## Đổi ảnh
Thay file `anh.jpg` bằng ảnh của bạn (giữ nguyên tên) là toàn bộ ảnh trên web tự đổi.
